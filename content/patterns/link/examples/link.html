リンクの例

この例について

下記の例は、リンクパターンの三つのバリエーションをデモンストレーションするものです。HTMLの a 以外の要素に対して、リンクの振る舞いを持たせる必要がある場合に、このリンクパターンは用いられます。

注記: 可能な限り HTML の a 要素を使用してリンクを作成してください。ブラウザは、ネイティブな HTML のリンクに対して、例えばリンク先を新しいウィンドウで開く、システムのクリップボードにリンク先 URL をコピーする、といった価値のある機能を提供します。
